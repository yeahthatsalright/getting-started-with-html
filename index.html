Notes on HTML

-Web Pages-

Web pages are designed to give information using a code of HTML to make users
able to see and access content by allowing browser to affectively communicate with
servers.
All of this can be found here.

-In Line vs. Block Elements-

In line lettering are things like "[br]" and "[span] that are used as containers for text
associated with HTML text with styling and other features.

Block elements examples would be things like "[p]" and [div] because they are used
as a container for in-line elements to alter the visual nature of the web page.

A good reference for this material can be found at this website.
